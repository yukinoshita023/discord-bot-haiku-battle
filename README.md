# 俳句バトル
## Discordサーバー「わくせい」の常駐型BOT

### 機能
- 専用チャンネルで2チーム対抗の俳句バトルを常時開催
- チーム名は花の名前からランダムに決定（🌸 vs 🌺）
- 5音 → 7音 → 5音 のリレー形式で投句し、チームの俳句を完成
- 完成した2句をリアクション投票で対決
  - 先に4票獲得したチームが即勝利
  - 初票から30分後（無投票の場合は6時間後）に票数の多い方が勝利、同票なら抽選
- 勝利チームのメンバーに俳句ポイントを付与（Firestoreに保存）
- 投句は1句25文字以内、画像・URL・メンション・絵文字は使用不可

### コマンド
- `/haiku_ranking` : 俳句ポイントのランキングTOP10を表示
- `/haiku_reset` : 進行中のバトルを強制終了して新しいゲームを開始（管理者専用）
